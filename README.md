# Ant Simulator

Estrutura modular para simulação simples de formigas em Pygame.

## Executar

1. Crie um virtualenv (opcional)
2. Instale dependências:
   pip install -r requirements.txt
3. Rode o jogo:
   python -m src.main

## Testes

pytest -q
